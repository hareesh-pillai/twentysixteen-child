== Description ==
This is the Twenty Sixteen Child theme. This theme features a full width template, without the sidebar. 
This theme is licensed under the GPL v2.0 or later license. :) Use it to make something cool.

For more information about the parent theme, i.e Twenty Sixteen please go to https://codex.wordpress.org/Twenty_Sixteen.

== Installation ==

1. In your admin panel, go to `Appearance -> Themes` and click the `Add New` button.
2. Click on the `Upload` button. Click on choose file, and choose twentysixteen-child.zip
3. Click on `Install Now`.
4. Click on the `Activate` button to use your new theme right away.
5. Note that the presence of the Twenty Sixteen theme in the theme directory is a must for this child theme to work.
